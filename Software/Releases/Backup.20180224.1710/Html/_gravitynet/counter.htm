Uso del Contador de Visitas

En su plan de alojamiento se ha instalado el cgi counter.exe dentro del directorio /cgi-bin. Se trata de un contador de visitas para su pagina web.



Características:

  • Lo puede insertar en una o varias páginas.
  • Visible o invisible.
  • Es posible definir diferentes dígitos o incluso crear sus propios dígitos.
  • Le permite definir numero de visitas iniciales, o modificar el numero de visitas en cualquier momento.
     


Funcionamiento:

El uso del contador es muy sencillo. Solo tiene que copiar el siguiente código html e insertarlo en su página/s web:

<img src="/cgi-bin/counter.exe?link=pagina1&style=pix2" alt="Contador">
 

Esta página ha sido visitada: veces

 

Propiedades:
En el código de arriba, puede ver dos parámetros:
  • link: El valor por defecto es "pagina1".
Si quiere insertar contadores en diferentes páginas, tendrá que modificar el valor de link para cada página.
Ej: pagina1 para la pagina principal, pagina2 para una de las paginas secundarias, etc..
  • style: Este valor define el estilo de dígitos que quiere mostrar. Dentro del directorio /cgi-bin/digits encontrará varios 
estilos de dígitos.

 

Archivos del contador:

  • counter.exe : Es el programa cgi compilado en C.
  • counter.dat : En este archivo de texto se guarda la información de visitas. Un ejemplo de counter.dat que almacena visitas de dos contadores seria el siguiente.
     
    2
pagina1 1504
pagina2 552

    La primera línea muestra el numero de contadores (2)
    Las líneas siguientes muestran el nombre del contador y el numero de visitas

     

  • counter.cfg : Es un archivo de texto donde se puede configurar el acceso externo al contador. No es necesario modificarlo. Tal y como esta por defecto, permite el uso del contador solo desde su pagina web.
  • /digits : directorio donde se guardan los dígitos. Dentro hay una carpeta /clearled, donde están los dígitos transparentes para poder instalar un contador invisible.
