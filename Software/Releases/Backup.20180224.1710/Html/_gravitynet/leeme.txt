
Información sobre los archivos instalados en su plan de alojamiento:
####################################################################


-default.htm: pagina que muestra el texto "en construccion". Debe sobreescribir esta pagina con su pagina de inicio.


-cgi-bin/counter.exe (contador de visitas): Abra el archivo counter.htm, aquí se describe como utilizar el contador. El contador, counter.exe, se encuantra en su directorio cgi-bin, aqui tambien esta ubicado el directorio /digits con los graficos para cada numero.


-aspmail.asp (formulario a email): abra el archivo aspmail.txt, aquí encontrará ayuda de como utilizar el componente aspmailer para el envío de formularios a email.
El archivo aspmail.htm es un ejemplo de formulario.
 


El directorio CGI-BIN tiene todos los permisos necesarios para ejecutar Perl, ISAPI y otras aplicaciones CGI, como precompilados ejecutables ".exe".
Debido a los permisos de este directorio, es recomendable que suba sus bases de datos Access aqui. 


La ejecucion de ASP, ASP.NET y PHP es posible desde cualquier directorio, incluso desde el raiz.
